Code & Tonic on GitHub

About 2 months ago, I launched a new website – Code & Tonic. I enjoy blogging but rarely get make time to sit down and do it. So, I had the idea to create a simple website where I could share sites, articles and products that I find awesome but don’t want to write an entire article about.

Tonight, I decided to make the code available on GitHub Even though I took an existing awesome WordPress theme and turned it into a Barley theme, I still felt it might help some people who are getting started with Barley.

You can head over to the GitHub repo and download it if you’d like. As always, feed back is appreciated.
